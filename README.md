Классическая аркадная игра Flappy Bird, переосмысленная с добавлением новых механик и улучшенным пользовательским опытом. Проект сохраняет дух оригинала, но предлагает больше вариативности и возможностей для прогресса.

🎮 Особенности игры
- Система бонусов: Во время полета на поле появляются специальные предметы (например, яблоки), которые можно собирать для получения дополнительных очков или временных улучшений.
- Кастомизация и сложность:
Выбор различных персонажей (скинов) для уникального визуального стиля.
Несколько уровней сложности, позволяющих адаптировать игру под свой навык — от новичка до эксперта.

- Статистика и прогресс: Встроенная история попыток, позволяющая отслеживать лучшие результаты и анализировать свой прогресс со временем.
